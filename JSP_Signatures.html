How to install
💻  Outlook Windows
1. Outlook → File → Options → Mail → Signatures → New → name it JSP Full

2. Click into the editor → Ctrl+A to clear → come back here, click inside your Full Signature box above → Ctrl+A → Ctrl+C

3. Paste (Ctrl+V) into Outlook editor → OK

4. Repeat for JSP Condensed using the condensed block

5. Set New messages → JSP Full  ·  Replies/forwards → JSP Condensed
🌐  Outlook Web (OWA)
1. Go to outlook.office.com → ⚙ Settings → View all Outlook settings → Mail → Compose and reply

2. Click inside your Full Signature box here → Ctrl+A → Ctrl+C

3. In OWA editor, delete existing content → Ctrl+V → Save
 
  Outlook Mac
1. Open this page in Safari (not Chrome — Safari keeps table formatting)

2. Click inside the Full Signature box → ⌘A → ⌘C

3. Outlook → Outlook menu → Preferences → Signatures → + → name it JSP Full → ⌘V to paste

4. Repeat for condensed → set both as defaults
📱  Mobile
1. Once saved in Outlook Web, your signature syncs automatically to the Outlook mobile app

2. Verify: open Outlook app → tap Compose → signature appears at the bottom

3. Not showing? Profile icon → Settings → Signature → enable it
